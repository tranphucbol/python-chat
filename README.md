## Tính năng 

- Tạo tài khoản (username/password, email,...)
- Chọn người trò chuyện (theo username hoặc theo email)
- Hiển thị lịch sử trò chuyện
- Trò chuyện (Chat)
- Kết bạn
- Chat trong nhóm
- Hiển thị trạng thái online/offline của người khác
- Hiển thị trạng thái tin nhắn (seen)
